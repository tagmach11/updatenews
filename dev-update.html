  • (관리자) 설정 > 일반에 로그인 유지시간 및 로그아웃 경고 팝업 시간 설정 기능 추가

  • (관리자) 통계 > 대시보드 차트 제목 우측에 총 인원 수 표시

  • (관리자) 과정·기수 화면에 날짜 선택 버튼 추가

  • (관리자) 과정 대시보드 > 목차구성의 '차시추가' 버튼을 마지막 차시 다음에 배치

개선사항

  • (관리자) 관리자 로그인 화면에서 '운영관리자' 버튼 삭제

  • (관리자) 과정 상세 > 학습자 노출 항목 명칭 수정 및 라디오 선택 시 설명 추가

오류수정

  • (사용자) 디자인 B 시안에서 질문하기 클릭 시 이전 로그인 페이지로 이동되는 오류 수정

  • (관리자) 과정 메뉴 분류표시 오타 수정

  • (관리자) 과정 > 수료현황 잘못된 컬럼표시 오류 수정

  • (관리자) 과정 대시보드 차시/구차시 버튼 명칭 오타 수정

2024년 11월 12일 릴리즈

신규기능

  • 로그인 화면 인증코드 입력 단계에 뒤로가기 버튼 추가

  • (관리자) 학습자/교수자 '엑셀등록' 버튼을 '일괄등록'으로 변경 및 아이콘 변경

  • 마이페이지 항목 간 간격 좁게 표시되는 오류 수정

  • 미수료 인원 마이너스(-) 표기 예외 처리
